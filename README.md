# project_euler_java

This project is an empty shell which you will fill with code to solve the problems from Project Euler.

# Getting Started

- Fork this repo into your own github account
- Start a Github Codespace 
- Install the extension "Extension Pack for Java" (vscjava.vscode-java-pack) if not already installed by devcontainer

# Solving a problem

- Open https://projecteuler.net/archives 
- Pick a problem (eg. 1) and open the page
- In your Github Codespace, create a file Problem1.java (use the number from your Project Euler problem)
- Add the following text as a starting point:
    ```
    public class Main {
        public static void main(String[] args) {
            // TODO solve the problem    
        }
    }
    ```
- Press the play button on the top right of the editor (or run `java Problem1.java` in the terminal)
- Update code and re-run until you solve the problem
- Commit your solution when done
- Start next problem
